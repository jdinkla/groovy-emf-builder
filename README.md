The EMFBuilder is a tool for programming with the Eclipse Modelling Framework with
Groovy. It was written in 2007.

See the web pages on EMF and UML2 on http://www.dinkla.net for details.

Thanks to laszlo.suto@gmail.com for providing a patch.
	
A jar and the dependencies are available for download on my website.

See http://www.dinkla.net/en/programming/groovy-emf.html
